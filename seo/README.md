Reference: [Google SEO Starter Guide](https://developers.google.com/search/docs/fundamentals/seo-starter-guide)

## Sitemap & Robots

- **Sitemap**: `/sitemap.xml` – Valid XML format, `Content-Type: application/xml`
- **Robots**: `/robots.txt` – Plain text, `Content-Type: text/plain`
- Ensure `NEXT_PUBLIC_APP_URL` is set to production domain (e.g. `https://waitlist.hyperkitlabs.com`)

## Best Practices (per Google)

1. **Titles** – Unique per page, clear, ≤60 chars; include primary keyword + brand.
2. **Meta descriptions** – Unique, concise, ≤155 chars; include primary keyword and CTA.
3. **Content** – Easy to read, unique, up to date, helpful, people-first.
4. **Images** – Descriptive alt text; place near relevant text.
5. **Links** – Descriptive anchor text; link to trusted resources.
6. **URLs** – Descriptive and readable.
7. **Avoid** – Keyword stuffing, meta keywords tag, excessive repetition.
